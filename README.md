BatchDeleteListview
===================
-----------------------------------
项目说明：<br />

  这是一个ListView批量删除的Demo,给需要的童鞋一个参考。
ListView有两种模式.即默认模式和编辑模式.当长点击某个条目的时候，ListView进入可以编辑的状态,就可以批量删除了.如图所示：<br />
